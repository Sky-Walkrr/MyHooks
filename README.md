# MyHooks

--

自用的 Xposed 模块，已适配[LSPosed](https://github.com/LSPosed/LSPosed)

- [x] 替换 MIUI 传送门的“翻译”按钮为“分享”（tested on v2.5.10）；
- [x] QQ dialogs 支持 cancelable；
